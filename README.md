# Blur-Activator-Deactivator

Blur Activator é um modulo que ativa o blur em aparelhos que não possuem Blur por padrão na rom, compatível com qualquer aosp A12+

Blur Deactivator é um módulo que desativa o blur do aparelho, recomendado para quem visa ter mais desempenho e uma rom mais fluída, ou para aqueles que não gostam tanto de ter blur na rom!
